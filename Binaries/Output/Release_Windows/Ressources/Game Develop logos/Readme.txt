Game Develop logos
2008-2012 Compil Games
-------------------

Game Develop's logos can be used in a game to specify that this game was created using Game Develop, as long as these logos stay unmodified and you do not state being the author of the logos.

Les logos de Game Develop peuvent être utilisés pour afficher votre soutien au logiciel, au sein de votre jeu ou sur votre site par exemple.
Vous pouvez les inclure librement tant que vous ne vous déclarez pas auteur de ces logos ou du logiciel et tant que les logos ne sont pas modifiés.
